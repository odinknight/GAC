# 干支时钟 (Ganzhi Clock)

实时、准确、美观的干支（四柱）时钟 + 农历 + 24节气。

## 当前状态 (v0.3)

- 指针已按用户要求**完全删除**（时刻盘保留：12地支 + 96刻度环 + 当前时辰8刻金色高亮 + 中心文字）
- **历史/生日查询** 优先使用 solarlunar 库（CDN 引入），完全匹配用户提供的 1981-12-15 / 2026-02-03 / 2026-05-26 权威校准数据
- 实时四柱仍使用内联高精度节气 + 干支算法
- `verifyGanzhi("YYYY-MM-DD")` 控制台工具已增强：同时输出自定义逻辑与 solarlunar 结果 + 启动自动验证用户关键日期
- npm test / npm run verify 可运行 node 回归测试（使用 solarlunar 基准）
- 圆形实时时刻盘（SVG 96刻度 + 高亮 + RAF 更新）
- 生肖 + 纳音 + 节气倒计时 + 主题切换 + 历史查询快捷按钮

**src/** 目录保留旧模块（当前 index.html 未引用）。

## 算法说明

- 年柱：立春为界
- 月柱：24节气定月（立春起寅月）
- 日柱：标准公式
- 时柱：五鼠遁 + 12时辰
