# 干支时钟 (Ganzhi Clock)

实时、准确、美观的干支（四柱）时钟 + 农历 + 24节气。

## 当前状态 (v0.4)

- **架构改进**：核心干支算法已提取到 `src/ganzhi.js`（纯函数 `getFourPillars`、`getLunar` 等），便于测试与未来重构
- 指针已按用户要求**完全删除**（时刻盘保留：12地支 + 96刻度环 + 当前时辰8刻金色高亮 + 中心文字）
- **历史/生日查询** 优先使用 solarlunar 库（CDN 引入），完全匹配用户提供的 1981-12-15 / 2026-02-03 / 2026-05-26 权威校准数据
- 实时四柱使用内联（现已模块化）的高精度节气 + 干支算法
- `npm test` 同时验证 solarlunar + 新核心模块 `src/ganzhi.js`
- `verifyGanzhi("YYYY-MM-DD")` 控制台工具仍可用
- 圆形实时时刻盘（SVG 96刻度 + 高亮 + RAF 更新）
- 生肖 + 纳音 + 节气倒计时 + 主题切换 + 历史查询快捷按钮

**已知技术债务**：
- 自定义 `getLunar` 仍包含分段校准 hack，部分日期（如 2026-02 附近）精度不足（推荐历史查询走 solarlunar）。
- `src/lunar.js` / `src/solarTerms.js` 为早期标准版，当前未被主程序使用。

## 算法说明

- 年柱：立春为界
- 月柱：24节气定月（立春起寅月）
- 日柱：标准公式
- 时柱：五鼠遁 + 12时辰
