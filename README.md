# 天干地支 + 星体动态时钟项目知识库
## 项目目标
开发一个动态时钟：
- **内圈/核心**：天干地支（Ganzhi）排列，精确到时辰（shi chen）和刻（ke），对应中国时区（CST UTC+8），底层使用格林尼治动态时间（UTC/TT）计算。
- **外围**：360度刻度，显示太阳、月亮、水星、金星、火星、木星、土星的实时天文位置（黄道经度 ecliptic longitude）和运行轨迹。
- 融合天文学精确计算 + 中西方星相学、易经占卜文化解读。
- 所有数据交叉验证自NASA JPL、Astropy、传统历法典籍。

### 时辰与刻的精确划分
- **时辰 (Shi Chen)**：12个，每2现代小时。对应地支。
  精确映射（中国标准时间 CST）：
  - 子时：23:00 - 01:00
  - 丑时：01:00 - 03:00
  - 寅时：03:00 - 05:00
  - 卯时：05:00 - 07:00
  - 辰时：07:00 - 09:00
  - 巳时：09:00 - 11:00
  - 午时：11:00 - 13:00
  - 未时：13:00 - 15:00
  - 申时：15:00 - 17:00
  - 酉时：17:00 - 19:00
  - 戌时：19:00 - 21:00
  - 亥时：21:00 - 23:00

- **刻 (Ke)**：传统1日=100刻，每刻约14.4分钟。现代常简化为每时辰8刻（每刻15分钟）。
  - 每个时辰分“初”（前半）和“正”（后半），或连续刻度。
  - 动态时钟建议：连续百分比或精确分钟到刻的映射。

**时干计算**：时辰地支固定，时干由日干推算（日干序号相关公式）。

## 3. 天体位置 - 天文学精确数据
### 对应关系（中西星相学融合）
- **木星 (Jupiter)**：木星 - 木行，东方，生长，易经关联木卦。
- **火星 (Mars)**：火星 - 火行，南方，热情/冲突。
- **土星 (Saturn)**：土星 - 土行，中央，稳定/限制。
- **金星 (Venus)**：金星 - 金行，西方，美丽/和谐。
- **水星 (Mercury)**：水星 - 水行，北方，智慧/沟通。
- **太阳 (Sun)**：阳，中心，生命力。
- **月亮 (Moon)**：阴，情绪、周期，与地支强关联（原月相观测）。

### 天文坐标
- **黄道经度 (Ecliptic Longitude)**：0°-360°，从春分点（白羊座0°）起算。所有行星近黄道运动。
- **实时位置**：使用Astropy / JPL DE / VSOP87 计算地心或日心黄经。
- **轨迹**：行星有顺行/逆行（retrograde），外行星轨迹缓慢，内行星快速。
- **动态**：每分钟/秒更新位置。轨迹可视化为弧线或预测路径。

**计算库**（已验证可用）：
- Astropy (coordinates.get_sun, get_moon, solar_system_ephemeris)
- 可选：astronomy-engine, skyfield (若可用)。

## 4. 融合文化解读（专家视角）
- **易经文化**：Ganzhi与八卦、河图洛书、五行生克相关。时钟可叠加卦象能量流。
- **占卜/星相**：当前Ganzhi + 行星位置 = 实时“天象”。例如，木星在某宫位影响“木”相关事务。西方占星黄道与中式结合（注意岁差 precession）。
- **天文学与形而上**：行星真实运动驱动“气”场。太阳月亮主导昼夜与潮汐/情绪；外行星长期周期对应大运。
- **西方对照**：黄道12宫 vs 地支12；行星统治 vs 五行属性。

## 5. 动态时钟设计建议
**可视化结构**：
- **中心**：当前北京时间 + Ganzhi 四柱（年月日时）或简化时+刻显示。字符旋转或高亮当前时辰。
- **内环**：12时辰 + 刻度细分（每时8-刻或连续）。
- **中环**：天干辅助或五行颜色。
- **外环 360°**：黄道圆，标记7天体当前位置（用符号或颜色：金木水火土 + 日月）。可添加轨迹线（过去/未来小时）。
- **交互**：实时更新（WebSocket或定时器），点击天体显示详情/占卜解读，切换历史时间模拟。
- **技术栈建议**：Python (Matplotlib/Plotly + Astropy) 原型；Web (D3.js + astronomy.js 或 backend Flask + Astropy)；App (Flutter or native)。
- **精度**：天文位置用高精度星历；Ganzhi用标准历法公式。支持闰秒/TT-UTC。

**原型实现路径**：
1. 计算当前UTC -> CST转换。
2. 计算当前Ganzhi (年/月/日/时 + 刻)。
3. 用Astropy获取各天体黄经。
4. 绘制极坐标图或Canvas圆环。
5. 动画更新。
